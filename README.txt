Для запуска тестов запустите последовательно 3 скрипта: gen.cmd, make.cmd и test.cmd
Скрипт gen.txt генерирует вспомогательные классы из файла грамматики JavaScript.g4
Скрипт make.cmd компилирует проект
Скрипт test.cmd запускает тесты из папки tests

Скрипт clear.cmd удаляет все сгенерированные файлы
